== Description ==

Do you send a newsletter based on content from your website using your Mailchimp account? Ever want to select a bunch of posts and have them appear in your next email campaign? If so, CampaignPress is the solution for you.  

== Installation ==

Installing "CampaignPress" can be done by searching for "CampaignPress" via the "Plugins > Add New" screen in your WordPress dashboard. 

1. Upon first install, you will be asked to select your MailChimp campaign list, frequency of your email/newsletter and a Page that you would like to use as the template for your email/newsletter (read below for more information on this).
2. After install, each Post will now have a section that allows you to select the Post for inclusion in the email/newsletter. Before each email/newsletter is sent, you 
3. Based on the frequency you select, the email/newsletter is sent to the specified campaign list. After the email/newsletter has been sent, all Posts that you indicated for inclusion are reset so that the next email/newsletter does not include the same content again.
